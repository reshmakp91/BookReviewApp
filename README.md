# 📚 BookReviewApp  

BookReviewApp is a **full-stack blog platform** where users can **post book reviews and comment** on others' reviews. Built with **React, Django, and REST API**, this project provides an interactive and engaging space for book lovers to share their thoughts and discover new reads.  

## 🚀 Features  

- 📝 **Post & Manage Reviews** – Users can write, edit, and delete book reviews.  
- 💬 **Comment on Reviews** – Engage in discussions by commenting on reviews.  
- 🔍 **Search Books & Reviews** – Easily find books and related reviews.  
- 🔑 **User Authentication** – Secure login & registration system.  
- 👤 **Edit Profile** – Users can update their profile details.  
- 📖 **Pagination** – Browse content smoothly with paginated results.


## 🛠 Tech Stack  

- **Frontend:** React, Bootstrap, CSS  
- **Backend:** Django, Django REST Framework, Python  
- **Database:** MySQL
- 
## 📌 Usage
- Open the frontend at http://localhost:3001/
- Access the backend API at http://localhost:8000/
